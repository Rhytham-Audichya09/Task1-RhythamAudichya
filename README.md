# To-Do List 

## Overview
This is a simple To-Do List application developed using Python. It allows users to manage their daily tasks efficiently by adding, viewing, and removing tasks.

## Features
- Add new tasks
- View all tasks
- Remove completed tasks
- Easy-to-use command-line interface

## Technologies Used
- Python 3

## How to Run
1. Install Python on your system.
2. Download or clone the project files.
3. Open the terminal in the project folder.
4. Run the program using:
 to_do_list.py

## Project Objective
The main objective of this project is to help users organize their tasks and improve productivity while demonstrating basic Python programming concepts.
